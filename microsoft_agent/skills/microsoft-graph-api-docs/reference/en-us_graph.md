## Services and features
### Users
  * [Access and manipulate user resources directly.](https://learn.microsoft.com/en-us/graph/azuread-users-concept-overview)


### Groups
  * [Enable user collaboration and integration across services.](https://learn.microsoft.com/en-us/graph/microsoft365-groups-concept-overview)


### Identity and access management
  * [Use Microsoft Entra methods to manage access and authenticate users.](https://learn.microsoft.com/en-us/graph/identity-network-access-overview)


### Security
  * [Connect Microsoft security products, services, and partners to streamline security operations.](https://learn.microsoft.com/en-us/graph/security-concept-overview)


### Microsoft Teams teamwork and communications
  * [Explore chat-based collaboration, meetings, calling, and enterprise voice features.](https://learn.microsoft.com/en-us/graph/teams-concept-overview)


### Outlook calendar and mail
  * [Control calendars such as creating an event.](https://learn.microsoft.com/en-us/graph/outlook-calendar-concept-overview)
  * [Access and manage messages and mail data.](https://learn.microsoft.com/en-us/graph/outlook-mail-concept-overview)


### Microsoft 365 Copilot connectors
  * [Explore Copilot connectors.](https://learn.microsoft.com/en-us/graph/connecting-external-content-connectors-overview)
  * [Build your first connector.](https://learn.microsoft.com/en-us/graph/connecting-external-content-build-quickstart)


### Microsoft Graph Data Connect
  * [Explore Microsoft Graph Data Connect.](https://learn.microsoft.com/en-us/graph/data-connect-concept-overview)
  * [Build a Data Connect application.](https://learn.microsoft.com/en-us/graph/data-connect-quickstart)


### All services and features
  * [Explore services for productivity, collaboration, people and workplace intelligence, education, and more.](https://learn.microsoft.com/en-us/graph/overview-major-services)
