The Cloud Adoption Strategy Evaluator assesses your strategy posture across distinct areas of the strategy methodology — identifying motivations, documenting expected business outcomes, evaluating financial considerations, and technical considerations in creating a business case. #### Azure Well-Architected Review
The Microsoft Azure Well-Architected Review tool generates a set of recommendations through a guided assessment based on the Microsoft Well-Architected Framework. This tool also has the ability to pull in Azure Advisor recommendations based on an Azure subscription or resource group. At the end of the assessment, there is an option to export these recommendations into a CSV file that can then be used to incorporate them into the operational process for the workload.
[ Start Assessment ](https://learn.microsoft.com/en-us/assessments/azure-architecture-review)
##  How assessments work
The Microsoft Assessments Procedure illustrates the continuity of assessments and how over time, customers can improve their score by following personalized recommendations based on their assessment results.
![](https://learn.microsoft.com/en-us/media/learn/assessments/icons/i_assessments_procedures.svg?branch=main)
###  Microsoft Assessments Procedure
![](https://learn.microsoft.com/en-us/media/learn/assessments/icons/i_start.svg?branch=main)
##### 1. Start
Choose an assessment that aligns with the business strategies you'd like to evaluate
![](https://learn.microsoft.com/en-us/media/learn/assessments/icons/i_about_you.svg?branch=main)
##### 2. About you
Provide answers to fundamental questions, which in return will narrow your content options
![](https://learn.microsoft.com/en-us/media/learn/assessments/icons/i_get_a_score.svg?branch=main)
##### 3. Get a score
Receive curated and personalized guidance that fits your specific scenarios
![](https://learn.microsoft.com/en-us/media/learn/assessments/icons/i_take_action.svg?branch=main)
##### 4. Take action
Review recommendations at your convenience to improve your score
![](https://learn.microsoft.com/en-us/media/learn/assessments/icons/i_recheck_score.svg?branch=main)
##### 5. Recheck score
Save your assessment progress through signing in and creating a milestone.
![](https://learn.microsoft.com/en-us/media/learn/assessments/icons/i_improve.svg?branch=main)
##### 6. Improve
Creating milestones will allow you to gradually improve your score and see your progress in real-time
